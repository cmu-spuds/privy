# Privy Frontpage

Minimal, research-lab inspired landing page for **Privy** — an AI-powered privacy risk assessment tool for AI product teams.

## Stack

- React + Vite
- Tailwind CSS (v4)
- Framer Motion
- Lucide React icons

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
